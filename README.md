# ❄️ myllena-emilly-FROZEN

> Uma aplicação interativa e visual com a temática de Frozen, desenvolvida utilizando tecnologias web fundamentais.

O projeto está publicado e pode ser visualizado diretamente no navegador através do link abaixo:
🔗 [Acesse o Projeto Aqui](https://github.io)

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

* **HTML5 (75%):** Responsável por toda a estrutura e marcação dos elementos da página.
* **CSS3 (16%):** Utilizado para a estilização, layout, cores e responsividade visual.
* **JavaScript (9%):** Aplicado para adicionar comportamentos dinâmicos e interatividade.

---

## 📁 Estrutura de Arquivos

O repositório está organizado de forma simples e direta:

* `index.html`: Arquivo principal contendo a estrutura da página.
* `style.css`: Folha de estilo contendo as regras de design visual.
* `script.js`: Arquivo de script para as lógicas de interatividade.
* `Frozen.webp`: Imagem otimizada utilizada como elemento visual da temática.

---

## 🚀 Como Executar Localmente

Caso queira baixar e rodar este projeto no seu computador, siga os passos abaixo:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd myllena-emilly-
   ```

3. **Abra o projeto:**
   Basta dar um duplo clique no arquivo `index.html` ou abri-lo utilizando a extensão *Live Server* no VS Code.
